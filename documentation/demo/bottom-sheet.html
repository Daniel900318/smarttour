Bottom Sheet


Bottom sheet is responsive meta component that is fixed at bottom of screen. It begins collapsed and toggleable in all viewports. To use it, html must has below basic markup and contain a component class - bmd-bottom-sheet.
Whereas there are two optional classes
bmd-bottom-sheet-list : Apply icon list style as below example
bmd-bottom-sheet-full-height : Usually use for more information of card component (see below example)
Besides, there is a toggle to trigger bottom sheet by two attributes in markup data-toggle="bmd-bottom-sheet" and data-target="#id" whereas dismiss is controlled by data-dismiss="#id"
