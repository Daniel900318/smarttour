$

Date and time input is using another jQuery plugin - bootstrap-material-datetimepicker which is customized for adding option "stateColor" and aligning flat button style to BMD. Please bear in mind to use customization version for BMD.

Date and Time input
code:
$('#inputDate').bootstrapMaterialDatePicker({ time: false, format: 'DD/MM/YYYY' });
code:
$('#inputTime').bootstrapMaterialDatePicker({ time: true, date: false, format: 'hh:mm A', stateColor: 'info' });
code:
$('#inputDateTime').bootstrapMaterialDatePicker({ time: true, format: 'DD/MM/YYYY - HH:mm', stateColor: 'success' });
