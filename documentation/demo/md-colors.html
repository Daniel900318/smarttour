MD color and utility classes

State color classes

Please note all color classes have !important to force overriding bootstrap rules.

Class name: bmd-{prop}-{state}
{prop} supports below CSS properties currently:
'text' - color
'bg' - background-color
'border' - border-color
'ink' - background-color (color of ripple)
{state} is 'default', 'primary', 'info', 'success', 'warning' or 'danger'

MD color classes

Class name: bmd-{prop}-{color}-[color-level]
{prop} supports below CSS properties currently:
'text' - color
'bg' - background-color
'border' - border-color
'ink' - background-color (color of ripple)
{color} is MD color listed as below table
[color-level] is MD color level and optional. It will use 500 if omitted.

Floating class

Class name: bmd-floating
Just makes an element has shadow.
