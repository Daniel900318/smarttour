Basic Usage


  <div class="row">
    <div class="col-lg-4">
      <!-- This is a target element to be transformed into and expanded on initial load -->
      <div id="morph-img" class="bmd-morph bmd-morph-expanded">
        <!-- display attribute must be block for <a> tag -->
        <a href="#" class="bmd-morph-close" style="display:block">
          <img class="img-responsive" src="images/work1.png">
        </a>
      </div>
    </div>
  </div>
  ..
  ..
  <!-- This is a morph FAB -->
  <a href="#" class="btn btn-primary bmd-fab bmd-morph-fab bmd-ripple bmd-floating"
    style="position:fixed;bottom:90px;right:20px" data-target="#morph-img">
    <span class="bmd-icon flaticon-mark1"></span>
  </a>

Options

Name Type Default Description
sticktarget boolean false Reserve column space for target element so that FAB always morph target at the same column. Otherwise, append to the end of row.
stickfab boolean false Only apply to FAB parent being <li>. Same as "sticktarget". Reserve original order for FAB so that FAB always home in the same order. Otherwise, append to the end of <ul>.
closemenu boolean false Only apply to FAB belonging to bmd-sidebar, bmd-bottom-sheet or bmd-fab-speed-dial-container. Menu will be auto closed when FAB left or homed.

Method

$("#example").morphFab(options)

Create a new instance of morph FAB with options

.morphFab("expand")

Transform into target element

.morphFab("close")

Revert to FAB state

Events

Event Type Description
expanded.bmd.morphFab This event is fired when target element is completely transformed (will wait for CSS transitions and animations to complete).
closed.bmd.morphFab This event is fired when FAB is completely reverted from target element (will wait for CSS transitions and animations to complete).
