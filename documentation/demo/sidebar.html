Class name Description
bmd-sidebar-active The sidebar menu is shown.
bmd-sidebar-collapse, fix, overlay Define sidebar menu behavior. One of them must be specified in markup.
bmd-sidebar-small Small sidebar menu is still shown when collapsed. The default width is 62px defined in LESS variable @small-menu-width.
bmd-sidebar-left, right Position the sidebar menu. Either one must be specified in markup.
bmd-allow-overflow To use it, please set together with bmd-page-container. Allow page contents overflow with full width of viewport. Some page contents will be invisible when sidebar menu is active and you may make to scrollable with overflow attribute of body.
bmd-sidebar-xs, sm, md, lg Predefined width: xs=150px, sm=200px, md=250px (default), lg=350px. Their LESS variables are @sidebar-xs-width, @sidebar-sm-width, @sidebar-md-width, @sidebar-lg-width
bmd-sidebar-brand Different css style from menu items.
bmd-sidebar-default, primary, info, success, warning, danger State color for sidebar menu background. bmd-sidebar-default is used if not specified.
bmd-sidebar-toggle-default, primary, info, success, warning, danger State color for sidebar toggle. bmd-sidebar-toggle-default is used if not specified.
